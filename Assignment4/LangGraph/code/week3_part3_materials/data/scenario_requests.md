Use these sample prompts to test agentic behavior.

"We need to ship one run for mobile users this week. Hard constraints are non-negotiable, but prioritize speed. Anything above 1.1s p95 hurts retention."

## Scenario B: Reliability-critical workflow
"Choose the run for a high-trust workflow. Hard constraints apply, but quality matters more than speed."

## Scenario C: General rollout
"Pick the best balanced run for general production traffic under current policy constraints."
