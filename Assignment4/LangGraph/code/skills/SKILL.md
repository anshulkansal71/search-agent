# Assignment Brief Skill

Use this skill when the user wants a polished assignment-style brief.

## Output format
Return markdown with these sections in order:
1. `## Context`
2. `## Key Findings`
3. `## Recommended Actions`
4. `## Risks / Open Questions`

## Writing rules
- Keep the output concise and evidence-focused.
- Prefer bullet points for findings and actions.
- Include dates when discussing recent events.
- If confidence is low, state assumptions clearly.
